# Resume Design: Junior Data Analyst — Hà Nội

**Date:** 2026-06-22
**Goal:** Career pivot CV from Unity Game Developer → Junior Data Analyst targeting Vietnamese local companies/startups in Hanoi.

---

## Context & Constraints

- **Language:** Vietnamese
- **Target market:** Vietnamese local companies and startups (not foreign-invested)
- **Current background:** ~4 years Unity Game Developer (Mirabo → Wingsmob → Zitga)
- **Pivot assets:** 3 data analysis portfolio projects, Coursera certifications, real analytics KPI experience from game live-ops
- **Top tools to feature:** SQL, Python, Power BI + AI agentic workflow

---

## Structure

```
1. Header + Tóm tắt
2. Kỹ năng
3. Dự án phân tích dữ liệu  ← hero section
4. Kinh nghiệm làm việc     ← reframed through data lens
5. Học vấn
6. Chứng chỉ
```

Rationale: Skills and portfolio projects appear before work history so the recruiter's first impression is "data analyst" not "game developer." Conventional enough for Vietnamese HR norms (still broadly chronological in work section), but front-loads data identity.

## Key Design Decisions

1. **Title is "Junior Data Analyst"** — not Unity anything. First impression = data.
2. **Portfolio URL in header** — recruiter can verify in 10 seconds.
3. **AI agentic workflow as a skill** — genuine differentiator; most junior DA candidates don't have this.
4. **Numbers lead every project bullet** — 1.6M records, p=0.0016, 90K users, 80% accuracy.
5. **Zitga reframe is honest** — Firebase/AdMob/AppLovin are real analytics platforms; the data work actually happened.
6. **Certifications at the bottom** — Google DA cert + Andrew Ng ML directly rebut "no formal DA training."
